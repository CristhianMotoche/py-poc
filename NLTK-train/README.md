# Learning NLP using the NTLK library

## Dependencies

For this project I used:

- Python 3.6.5
- Python Package Manager (pip) 18.0

Start virtualenv:

```shell
$ cd NTLK-train/
$ virtualenv .
$ source bin/activate
```

Install `pypi` dependencies:

```shell
$ which python
$ which pip
$ pip install -r requirements.txt
```
